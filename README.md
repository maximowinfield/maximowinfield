# 🚀 Featured Projects

## Palworld Server Manager

A Windows-based management and automation platform for operating a self-hosted Palworld dedicated server.

The project combines an **ASP.NET Core web dashboard** with a collection of **PowerShell administration scripts**, providing both browser-based moderator controls and lower-level server automation.

**Tech:** C#, ASP.NET Core Razor Pages, PowerShell, HTML, CSS, JavaScript, Palworld REST API, SteamCMD, Windows Task Scheduler

### Highlights

- Secure moderator login with permission-based controls
- Browser-based server management dashboard
- One-click world save, restart, and server announcement actions
- Palworld REST API integration
- Automated timestamped ZIP backups
- Configurable backup-retention cleanup
- Guarded backup restoration
- Server process and UDP-port health checks
- Graceful stop and restart workflows
- SteamCMD server updates
- Windows startup-task installation and removal
- Centralized configuration validation
- Secret-safe example configuration
- Responsive interface for desktop and mobile use

👉 **Source Code:** [github.com/maximowinfield/palworld-server-manager](https://github.com/maximowinfield/palworld-server-manager)

## Job Application Tracker

A full-stack application for organizing and tracking job applications throughout the hiring lifecycle.

The project was designed to replace fragmented spreadsheets and notes with a centralized system supporting authentication, filtering, application status management, and secure document handling.

**Tech:** React, TypeScript, C#, .NET 8, Entity Framework Core, SQLite, JWT, AWS S3

### Highlights

- User registration with securely hashed passwords
- JWT-based authentication
- Create, read, update, and delete application workflows
- Search, filtering, and status tracking
- Secure document uploads using Amazon S3 presigned URLs
- RESTful backend endpoints
- Persistent relational data storage
- Responsive frontend interface
